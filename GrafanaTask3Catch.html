Grafana - Task 3 - Catch the object

Catch the object game

Fang så mange sensorverdier som mulig innen 60 sekunder ved hjelp av venstre og høyre piltast

Grønne verdier er akseptable og skal fanges, mens røde verdier representerer støy og bør unngås

Gyldige verdier: Temperatur: 9–27°C, Fuktighet: 40–60 %RH, Vibrasjon: 0–5 mm/s

Ugyldige verdier inkluderer "ERROR", "NaN" og ekstreme målinger

Vær obs på at noen ugyldige verdier(Røde) kan være makert som gyldige (Grønne)

Det er +3 poeng for å fange riktige objekter og -1 poeng for å fange feil objekter

Resultatet vil bli visualisert som kakediagram og stolpediagram. Der brukeren kan analysere resultatet
